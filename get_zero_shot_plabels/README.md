# Obtaining zero-shot pseudo-labels from foundation models.

This directory contains following zero-shot inference classification examples (to use as zero-shot pseudo-labels in ZeroMatch training):
- [cv_gpt](./cv_gpt): Use GPT-4.1 model with CIFAR100, Flowers102, and Resisc45 dataset
- [nlp_gpt](./nlp_gpt): Use GPT-4o, Llama-3.3-70B model with AG news, Yahoo Answers, and Amazon Review dataset
- [cifar_clip](./cifar_clip): Use CLIP model with CIFAR100 dataset
- [framed_clip](./framed_clip): Use CLIP model with Flowers102, and Resisc45 dataset
- [nlp_flan](./nlp_flan): Use FLAN-T5 model with AG news, Yahoo Answers, and Amazon Review dataset
